MRB.sim


Mekong River Basin water pollution data simulator

Simulate economic climate and save results to publish

Site uses

This website uses data to genrate water quality data that could be used for economic purposes. It is a simulator that uses data from the Mekong River Basin to generate water quality data

Site functionality

Users can simulate water quality data and save the results to publish. The results can be saved privately so only the user can see them or they can be saved publicly so all users can see them

There are 4 regions that can be selected in the create area and their water variables can be changed. If you click simulate on the navbar then either you can search the public templates and choose one or you can press the green plus in the bottom right hand corner to create a new template.
